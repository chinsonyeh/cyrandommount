# CYRandomMount

CYRandomMount 是一個魔獸世界插件，通過動態巨集系統智能管理坐騎召喚。它根據您當前的區域（飛行、地面或特殊區域如幽坑城）自動選擇並更新隨機坐騎，盡可能避免重複以確保坐騎多樣性。插件支援角色特定和帳號共用坐騎偏好設定，通過直觀的設定面板提供靈活的配置選項。

## 功能

- **多語言支援：** 選項 UI 會自動以您的遊戲客戶端語言顯示（繁體中文、簡體中文、法文、德文、西班牙文、葡萄牙文、俄文、韓文、日文和英文）。
- **自動巨集更新：** 創建並更新名為 `CYRandomMount` 的巨集，以召喚適合您當前區域的隨機選定坐騎（飛行、地面或特殊區域如幽暗礦坑）。
  **巨集更新模式：**
   - **模式 1：** 每次按下巨集按鈕時，巨集將更新並選擇新的隨機坐騎（推薦）。
   - **模式 2：** 巨集將在固定間隔自動更新並選擇新的隨機坐騎（刷新時間）。
- **坐騎多樣性保證：** 選擇下一個隨機坐騎時，如果有多個坐騎可用，插件會從池中排除當前選定的坐騎，確保每次召喚時獲得不同的坐騎。
- **角色特定坐騎清單：** 在角色特定坐騎清單或帳號共用清單之間選擇，以最大化跨角色的靈活性。
- **喜愛坐騎選擇：** 從您收集的坐騎中選擇哪些飛行和地面坐騎包含在隨機選擇中。使用「全選」和「全不選」按鈕快速選擇。
- **自訂刷新間隔：** 設定巨集更新以反映您當前區域飛行能力的頻率（5-30 秒）。
- **可拖拽圖示：** 選項 UI 中的可拖拽圖示，可以拖拽到您的動作列以快速存取。
- **設定面板：** 通過遊戲內設定介面輕鬆配置您的偏好。

## 如何使用

1. **安裝插件：**
   - 將 `CYRandomMount` 資料夾放置在您的 `path_to\World of Warcraft\_retail_\Interface\AddOns` 目錄中。
   - 建議透過 `CurseForge` 搜尋 `CYRandomMount` 及安裝

2. **遊戲內設定：**
   - 在聊天中輸入 `/cyrandommount` 或 `/cyrm` 以開啟設定面板。
   - **選擇清單模式：** 選擇使用角色特定坐騎清單或帳號共用清單。
     - **角色特定：** 每個角色維護自己的坐騎偏好。
     - **帳號共用：** 所有角色共用相同的坐騎偏好（推薦新角色使用）。
   - **選擇坐騎：** 從您收集的坐騎中選擇您喜愛的飛行和地面坐騎。
   - **配置更新模式：** 選擇巨集如何更新：
     - **模式 1（推薦）：** 每次使用時更新巨集以獲得立即多樣性。
     - **模式 2：** 在設定間隔自動更新巨集。
   - **設定刷新間隔：** 調整在定期模式下巨集更新的頻率（5-30 秒）。

3. **使用巨集：**
   - 插件自動創建名為 `CYRandomMount` 的巨集。
   - 要使用巨集，將其從巨集 UI 拖拽到您的快捷列。或者，為更快速存取，直接從選項 UI 拖拽圖示到您的快捷列。
   - 使用時，它將召喚適合您當前區域的隨機坐騎。

## 注意事項

- 只有已收集的坐騎會出現在選擇清單中。
- 巨集會根據您的區域、刷新間隔和選定的更新模式自動更新。
